# Prework Study Guide Webpage

## Description

This is a study guide created for the UT Coding Bootcamp. It will be used to host notes on HTML, CSS, Git, and Javascript, and was created using those languages and systems. It served as a way to introduce the basics of web development, and can be referenced in the future.

## Installation

N/A

## Usage

This webpage can be used by viewing each topic and the corresponding notes displayed under them. Additionally if you right click, and inspect you can see the javascript running. Feel free to refresh the page multiple times to see the javascript select a different random topic.
