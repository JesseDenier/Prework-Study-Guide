Prework Study Guide

HTML

  • HTML is the structure or frame of a webpage.
  • HTML is created first.
  • The head element contains information about the webpage.
  • The information in the head element is metadata used by search enginges not users.
  • The body element represents the visible content shown to the user.
  • All elements within the HTML are surrounded by tags signifying what they are.

CSS

  • CSS is the styling of a webpage.
  • CSS defines the colors, aesthetics, visual layout, and organization.
  • A selector defines the elements to which the rules apply.
  • A declaraction lists a rule.
  • A property is what aspect the rule applies to.
  • A value is the specifics of that property.
  • Margin adds space to the outside of an element.
  • Padding adds space around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git pull origin main: pulls the latest main branch
  • git checkout main: moves you to the main branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git commit -m "example": attaches a message to our commit
  • git push origin feature/example: pushes the feature example to Github
